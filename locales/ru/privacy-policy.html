Политика конфиденциальности

Дата вступления в силу: 28 марта 2026 г.

Igor Vedeneev («Поставщик услуг», «мы», «нас») разработал приложение ReadLater («Приложение») как условно-бесплатный сервис. Эта страница информирует вас о нашей политике в отношении сбора, использования и раскрытия персональных данных при использовании Приложения.

1. Данные, которые мы собираем

  • Данные учётной записи — имя и адрес электронной почты, предоставленные через Apple или Google Sign-In
  • Данные закладок — URL-адреса, заголовки, описания и миниатюры, которые вы сохраняете
  • Данные об использовании — серия чтений, статистика прочитанных/сохранённых материалов, структура папок и тегов
  • Токены push-уведомлений — для отправки напоминаний и дайджестов
  • Информация о подписке — управляется через RevenueCat
  • Отчёты об ошибках и информация об устройстве — собираются через Firebase Crashlytics для повышения стабильности
  • Журналы сервера — журналы ошибок в Google Cloud Run для отладки; они не содержат личной идентификационной информации

2. Как мы используем ваши данные

  • Для предоставления и улучшения Приложения
  • Для организации ваших закладок с помощью функций искусственного интеллекта
  • Для отправки напоминаний и еженедельных дайджестов, на которые вы подписались
  • Для обработки платежей по подписке
  • Для диагностики сбоев и устранения ошибок

3. Сторонние сервисы

Приложение использует следующие сторонние сервисы, которые могут получать ваши данные:

  • Google Firebase (Firestore, Storage, Crashlytics, Cloud Messaging) — Политика конфиденциальности
  • RevenueCat — управление подписками — Политика конфиденциальности
  • OpenAI — ИИ-обработка заголовков, описаний и URL-адресов закладок — Политика конфиденциальности
  • Anthropic — ИИ-обработка заголовков, описаний и URL-адресов закладок — Политика конфиденциальности
  • Google Cloud Run — хостинг серверной инфраструктуры

Когда данные ваших закладок передаются в OpenAI или Anthropic для ИИ-обработки, передаются только URL-адрес, заголовок и описание. Данные учётной записи с этими сервисами не передаются.

4. Хранение и передача данных

Ваши данные хранятся на серверах Google Cloud, расположенных во Франкфурте, Германия (europe-west3). Миниатюры изображений хранятся в Firebase Storage.

Некоторые данные могут передаваться в Соединённые Штаты при обработке OpenAI, Anthropic или RevenueCat. Такие передачи осуществляются с соблюдением надлежащих мер защиты в соответствии с GDPR.

5. Ваши права по GDPR

Если вы находитесь в Европейской экономической зоне, у вас есть следующие права:

  • Доступ — запрос копии ваших персональных данных
  • Исправление — запрос исправления неточных данных
  • Удаление — удалите свою учётную запись и все связанные данные непосредственно в Приложении через Настройки → Учётная запись → Удалить учётную запись
  • Переносимость — запрос ваших данных в машиночитаемом формате
  • Возражение — возражение против обработки ваших данных

Для реализации любого из этих прав свяжитесь с нами по адресу getmaxxn@gmail.com.

6. Хранение данных

Ваши данные хранятся до тех пор, пока ваша учётная запись активна. При удалении учётной записи все персональные данные безвозвратно удаляются с наших серверов в течение 30 дней.

7. Конфиденциальность детей

Приложение не предназначено для детей младше 13 лет. Мы намеренно не собираем персональные данные детей до 13 лет.

8. Push-уведомления

Приложение может отправлять push-уведомления для напоминаний и еженедельных дайджестов. Вы можете отключить уведомления в любое время в настройках устройства.

9. Аналитика

Мы можем собирать анонимизированную аналитику использования, чтобы понять, как используется Приложение, и улучшить его. Эти данные не позволяют идентифицировать вас лично.

10. Изменения в настоящей политике

Мы можем периодически обновлять настоящую Политику конфиденциальности. Мы уведомим вас о существенных изменениях, опубликовав новую политику в Приложении. Продолжение использования Приложения после изменений означает принятие новой политики.
